## ğŸš€ Quick Start

This project is a client-side only application. There's no complex build process or server setup required.

### Prerequisites
-   A modern web browser (e.g., Chrome, Firefox, Safari, Edge).

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Palomino128/GastoSemanal.git
    cd GastoSemanal
    ```

2.  **Open the application**
    Simply open the `index.html` file in your preferred web browser.
    ```bash
    # For macOS
    open index.html

    # For Windows
    start index.html

    # For Linux
    xdg-open index.html
    ```

    Alternatively, you can manually navigate to the `index.html` file in your file explorer and double-click it.
